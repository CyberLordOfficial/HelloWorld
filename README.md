# HelloWorld

A simple C++ project demonstrating a basic "Hello World" application.

## Description
This project is a foundational C++ exercise. It uses a custom header file (`Main.h`) and a source file (`HelloWorld.cpp`) to print "Hello World!" to the console.

## Getting Started

### Prerequisites
* A C++ compiler (such as MSVC included with Visual Studio).
* Visual Studio 2022 or equivalent IDE like Visual Studio Code.

### Building the Project
1. Open the solution file (`.sln`) in Visual Studio.
2. Ensure your build configuration is set to **Debug** or **Release**.
3. Press **F5** or click the **Local Windows Debugger** button to build and run the application.

## Usage
The program will launch a console window and display:
`Hello World!`
